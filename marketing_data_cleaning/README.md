# Bank Marketing Data Cleaning Project

### **Project Description**
This project focuses on cleaning and restructuring a raw dataset from a recent marketing campaign. The goal is to transform messy data into a
structured format that adheres to a specific shema, emabling it to be seamlessly imported into a PostgreSQL database for future campaign analysis. 

### **Key Features**
- **Data Cleaning:** Handle missing values, normalize categorical strings (e.g., replacing "." with "_"), and ensure data consistency.
- **Feature Engineering:** Convert time-based data into unified datetime objects and encode categorical respnses into specific formats. 
- **Database Normalization:** Split the monolithic flat file into three relational tables to ensure efficient database storage and scalability.
